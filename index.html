World Wide Web - Collection of HTML Documents.
(Web invented in 1990s. Has about 30 billion pages).

TheMajor Pieces/Concepts of the web are best illustrated by this picture:

WEBPAGE SUMMARY:
- A webpage is a text document written in a language called HTML. Browsers read these documents and then
interpret & display them.

HTML - HyperText Markup Language.
- HTML Documents make up the web. Considered the "heart" of the web.

HTML is made up of the following:

1) Text Content - What you see.
2) Markup - What it looks like.
3) Reference to other Docs - e.g. Images & Videos.
4) Links to other Pages

HTML Markup:
< Name > Contents < /Name > ------ **This is known as an ELEMENT** (everything within a set of opening
and closing tags).
** < Name > = OPENING TAG
**< /Name > = CLOSING TAG

Tags are part of basic HTML vocabulary. An HTML tag is always contained within angled brackets.
Most Tags have an Open Tag and Closing Tag. Some examples are:

B Tag - < b > contents < /b >
b = BOLD (the contents will always appear in bold).

Em Tag - < em > contents < /em >
em = EMPHASIS (the content will always appear in italics).

Some Tags do not require a closing tag. These are called Void Tags. Examples are:

< img > = Image Tag
< br > = Break Tag

HTML ATTRIBUTES - A Property of an HTML Element.

- An HTML Attribute looks like the following:
< Tag Attr = "Value" > Contents < /Tag>

An example of a Tag that uses an Attribute is the Anchor Tag.
Anchor Tag - < a >
- Anchors are for making links.

FULL EXAMPLE: < a href = www.reddit.com > derp < /a >
**href = Attribute Name
**www.reddit = Value (In This Case a URL)
**derp = contents

IMAGE TAG ( < img >)
< img src = "url" alt = "text" >
** alt = text is required (text gets displayed when image does not load).

WHITE SPACE: Text entered on multiple lines are all entered into a single space. To force it to have multiple
lines, we use the tag < br > (Break Tag).

- Another way of using line breaks is to use the tag < p >
< p > stands for paragraph and looks like this:
< p > content < /p >

INLINE / BLOCK

Why do we have two ways for making new lines?

ANSWER:

< br > = Inline -- Says where to end the line and have the remaining text wrap to the next line
< p > = Block -- Makes invisable box around text that can have height and width

SPAN & DIV ELEMENTS
< Span > text < / Span > - Inline
< Div > text < /Div > - Block

- You can attach different styles to them to adjust behaviors of how they display.

OTHER IMPORTANT NOTES TO TAKE AWAY FROM THIS LESSON:

* How Coding Works - Coding happens when programmers write text in a language that a computer can
understand. The computer can then follow the instructions the programmer wrote.

* Computers Are Stupid - Programmers need to write exactly the way a computer understands (also known
as writing with the correct "Syntax"). For example, if you forget to close a < b > tag, the computer wont be
able to figure out what you had intended to make bold. This "Stupidicity" can be very frustrating, but it also
gives programmers incredible power: if you know how to talk to a computer than you can tell it to do anything
you want.

*Programmers Can't Remember Everything - There are too many details to keep everything in your head. If
you forget how to make text italic in HTML, you can always just look it up.
